# Farm Life

Farm Life is a cozy browser farming game prototype built with React and TypeScript. The project focuses on grid-based play, crop loops, and lightweight session management.

## Highlights

- Farming-oriented browser gameplay
- React state-driven UI and game flow
- Fast local iteration with Vite

## Stack

- React
- TypeScript
- Vite

## Quick start

```bash
npm install
npm run dev
```

Build and preview:

```bash
npm run build
npm run preview
```
